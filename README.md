# Task1  (Quiz project)

#### how to install this project



- ```
  # create venv
  py -m venv name_venv
  ```

  

- ```
  cd name_venv
  ```

  

-  

  ```
  Scripts\activate      (in windows)
  ```

  

- ```
  pip install Django 
  ```

  

-  

  ```
  pip install -r requirements.txt 
  ```
